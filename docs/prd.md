## 1. 项目概述

### 1.1 项目背景
「第二曲线」是成都本地的AI创客社区，旨在链接技术人、产品经理、创业者，共同探索AI应用和创业机会。

### 1.2 项目目标
- 建立社区品牌官网，展示社区形象
- 提供知识分享和案例展示平台
- 实现社区成员资源对接
- 支持后台内容管理
- 实现用户登录和权限管理

### 1.3 目标用户
- 成都本地AI技术人员
- AI产品经理
- AI创业者
- 对AI感兴趣的爱好者

## 3. 非功能需求

### 3.1 性能要求
- 首屏加载时间 < 2秒
- 页面切换响应时间 < 500ms
- 支持1000+并发用户

### 3.2 安全要求
- 用户密码加密存储
- 防SQL注入
- 防XSS攻击
- CSRF防护
- 敏感操作二次验证

### 3.3 可用性要求
- 响应式设计，支持PC和移动端
- 支持主流浏览器（Chrome、Firefox、Safari、Edge）
- 界面简洁，操作直观

### 3.4 可维护性要求
- 代码结构清晰，注释完整
- 模块化设计
- 有完整的技术文档
- Git版本控制

## 4. 数据模型设计

### 4.1 用户表（users）
| 字段名 | 类型 | 说明 | 约束 |
|--------|------|------|------|
| id | uuid | 用户ID | 主键 |
| email | varchar | 邮箱 | 唯一 |
| phone | varchar | 手机号 | 唯一 |
| password | varchar | 密码（加密） | 非空 |
| nickname | varchar | 昵称 | 非空 |
| avatar | varchar | 头像URL | |
| role | enum | 角色 | guest/member/core/admin |
| status | enum | 状态 | active/disabled |
| created_at | datetime | 创建时间 | |
| updated_at | datetime | 更新时间 | |

### 4.2 用户资料表（user_profiles）
| 字段名 | 类型 | 说明 |
|--------|------|------|
| id | uuid | 资料ID |
| user_id | uuid | 用户ID（外键）|
| bio | text | 个人简介 |
| skills | json | 技能标签 |
| looking_for | text | 寻找什么 |
| wechat | varchar | 微信号 |
| website | varchar | 个人网站 |

### 4.3 案例表（cases）
| 字段名 | 类型 | 说明 |
|--------|------|------|
| id | uuid | 案例ID |
| title | varchar | 标题 |
| category | varchar | 分类 |
| description | text | 描述 |
| content | text | 详细内容 |
| author_id | uuid | 作者ID（外键）|
| tags | json | 标签 |
| is_published | boolean | 是否发布 |
| created_at | datetime | 创建时间 |
| updated_at | datetime | 更新时间 |

### 4.4 成员表（members）
| 字段名 | 类型 | 说明 |
|--------|------|------|

### 4.5 活动表（events）

## 5. 界面设计

### 5.1 设计原则
- 简洁现代的设计风格
- 以黑白灰为主色调
- 品牌色点缀（橙色）
- 大量留白，呼吸感强
- 清晰的信息层级

### 5.2 设计规范
- 主色：#111827（深灰）
- 品牌色：#f97316（橙色）
- 圆角：12px-24px
- 字体：Inter / 系统默认无衬线字体
- 间距：8px基数
