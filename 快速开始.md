### 2. 部署上线（免费！）

推荐用 **Vercel** 部署，完全免费：

```bash
# 1. 先进入项目目录
cd /Users/doris/secondcurve-website

# 2. 初始化 Git（如果还没有）
git init
git add .
git commit -m "Initial commit"

# 3. 去 GitHub 创建一个新仓库，然后推上去
git remote add origin https://github.com/你的用户名/secondcurve-website.git
git branch -M main
git push -u origin main

# 4. 去 https://vercel.com 导入这个仓库，一键部署！
```
